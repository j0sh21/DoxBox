EN: On Linux, open a terminal in the directory containing install.sh and requirements.txt and run:
chmod +x install.sh
./install.sh

On Windows, open the Folder containing install.bat and requirements.txt and double click. 
Or opan a command prompt in the directory containing install.bat and requirements.txt and run:
install.bat


PT: No Linux, abra um terminal no diretório que contém install.sh e requirements.txt e execute:
chmod +x install.sh
./install.sh

No Windows, abra a pasta que contém install.bat e requirements.txt e clique duas vezes. Ou abra um prompt de comando no diretório que contém install.bat e requirements.txt e execute:
install.bat


DE: Auf Linux öffnen Sie ein Terminal im Verzeichnis, das install.sh und requirements.txt enthält, und führen Sie aus:
chmod +x install.sh
./install.sh

Auf Windows öffnen Sie den Ordner, der install.bat und requirements.txt enthält, und doppelklicken Sie darauf.
Oder öffnen Sie ein Eingabeaufforderungsfenster im Verzeichnis, das install.bat und requirements.txt enthält, und führen Sie aus:
install.bat
